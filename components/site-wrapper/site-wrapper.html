.site-wrapper

A way to wrap your entire site in order to avoid using the <body> element for things like display: grid; and to avoid potential 3rd-party plugin conflicts.

Can be useful when using grid for the main structural layout of your site. For example, this is one way to keep the site's footer on the bottom of short pages with minimal content.

HTML


<body>
  
  <div class="site-wrapper>

    <header class="site-header"></header>

    <main class="site-main"></main>

    <footer class="site-footer"></footer>

  </div>

</body>

CSS


/* site wrapper */
.site-wrapper {
    display: grid;
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
      "top"
      "mid"
      "bottom";
    position: relative;
    min-height: 100vh;
}

.site-header {
    grid-area: top;
}
.site-main {
    grid-area: mid;
}
.site-footer {
    grid-area: bottom;
}
